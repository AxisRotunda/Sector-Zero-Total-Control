## [3.2.0] - 2026-01-29
### Added
- **Hierarchical World System**: Introduced `WorldGraph` and `ZoneManager` for structured level progression.
- **Rift Network**: Added Waypoints and Personal Rifts for fast travel.
- **Advanced Codex**: Bestiary and Location Discovery systems.
- **Propaganda System**: Renderers for Banners and Holographic Signs.

## [3.1.0] - 2026-01-15
### Changed
- **Persistence Upgrade**: Migrated from `localStorage` to `IndexedDB` for larger world state snapshots.
- **Zoneless Refactor**: Removed `zone.js` dependency for improved Angular performance.

## [3.0.0] - 2026-01-01
### Added
- **Sector Zero: Total Control**: Initial release of the 3.0 rewrite.
- **Isometric Pipeline**: Custom canvas rendering engine.
- **Signal State**: Fully reactive Signal-based architecture.

If you have the repository cloned locally, run the following command to generate a full text log of the project history:

```bash

```

This will create a `project_history.txt` file in your root directory containing every change since the project's inception.
